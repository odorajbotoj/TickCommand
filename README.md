# TickCommand

Controls or queries the tick status of the game.

## Install

+ use lip.

```bash
lip install github.com/odorajbotoj/TickCommand
```

## Usage

+ `tick query` 查询mspt
+ `tick freeze|reset` 冻结/重置tick
+ `tick rate <float>` 调整tick速率
